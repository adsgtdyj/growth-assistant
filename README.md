# 🎯 AI成长助手 — 习惯打卡工具

> 一个移动端风格的单页 HTML 应用，帮助用户追踪习惯打卡、查看数据趋势，并配备 AI 对话助手。
>
> **⚠️ 当前状态：功能基本可用，AI 对话模块仍在迭代中。**

---

## 📋 项目说明

| 项目 | 内容 |
|------|------|
| 开发形态 | **单文件 HTML 应用**，无需构建工具、无需安装依赖 |
| 技术栈 | 原生 HTML + CSS + JavaScript（零框架、零依赖） |
| 数据存储 | 浏览器 localStorage（刷新不丢失） |
| 设计规格 | 移动端 375×812 设计（iPhone 类），响应式居中 |
| 当前版本 | v13 |

## ✨ 功能模块

| 模块 | 状态 | 说明 |
|------|------|------|
| 📋 **习惯管理** | ✅ 基本可用 | 创建/编辑/删除习惯，选择图标、颜色、打卡频率 |
| ✅ **今日打卡** | ✅ 基本可用 | 一键快速打卡、打卡弹窗添加备注、撤销打卡 |
| 📅 **日历视图** | ✅ 基本可用 | 按月查看打卡记录，点击日期查看详情 |
| ⏰ **提醒设置** | ✅ 可用 | 分时段提醒（早起、午休、晚间），提醒关联习惯 |
| 📊 **数据分析** | ✅ 基本可用 | 累计打卡/最长连胜/完成率 KPI、7 天趋势图、各习惯完成率对比、AI 洞察 |
| 👤 **个人中心** | ✅ 可用 | 头像上传（相册）、昵称/签名编辑、勋章墙、AI 语气切换 |
| 🔍 **习惯详情** | ✅ 基本可用 | 月历打卡展示、点击日期查看备注、补签功能 |
| 🤖 **AI 助手** | 🚧 **待调整** | 基础对话能力可用，意图识别和回答质量还在优化中 |

## 🚀 快速开始

1. 直接用浏览器打开 `mvp-app-v13.html`
2. 应用会**自动加载示例数据**（包含健身训练、阅读学习、每日饮水 3 个示范习惯和一段打卡记录）
3. 首次体验建议：
   - 首页 → 点击 `+` 号创建新习惯
   - 首页 → 点击习惯卡片完成今日打卡
   - 底部 Tab → 切换到"分析"查看数据看板
   - 底部 Tab → 切换到"助手"体验 AI 对话

## 🧠 设计要点

- **数据模型**：`habit`（习惯）和 `checkin`（打卡记录）两个核心结构，存储在 localStorage，key 为 `habit_checkin_app_v13`
- **AI 引擎**：基于关键词意图匹配的规则系统，支持温和/默认/毒舌 3 种语气
- **全量刷新**：`refreshAll()` 统一渲染所有页面，数据变更后自动更新
- **离线可用**：不依赖后端服务，纯浏览器端运行
